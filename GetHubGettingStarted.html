Getting your machine ready?

What you need

  1. Download Git from https://git-scm.com/downloads
  2. An editor Visual Studio Code
  3. GitHub account

Setting up git on your windows machine

  1. download and install git on your machine
  2. git config --global user.name "mreddy"
  3. git config --global user.email "t.mallikarjunreddy@gmail.com
  4. git config --edit --global
  5. git config --global "core.editor "code --wait --new-window" : code is the bash command to open VSCode --wait is for the cursor to wait till the VSCode window is closed

Using SSH

  1. HTTPS: Requires username and password
  2. SSH: Easier to work with
  3. Generating SSH Keys
  4. Connecting to GitHub using SSH
  5. at root dir ls -al ~/.ssh to get the list of SSH keys
  6. Create SSH Key: ssh-keygen -t rsa -b 4096 -c "t.mallikarjunreddy@gmail.com"
  7. verify SSH agent running: eval $(ssh-agent -s)
  8. add SSH key: ssh-add ~/.ssh/id_rsa
  9. ssh -t git@github.com
