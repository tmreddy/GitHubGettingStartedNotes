An Overview of Git and GitHub

What is Git?

  1. Popular source control system
  2. Distributed system
  3. Free and open-source

Why use Git?

  1. Fast
  2. Disconnected
  3. Powerful yet easy
  4. Branching
  5. Pull requests

Why Not Git?

  1. Different
  2. Learning curve
  3. Tools
  4. Binary files

What is GitHub?

  1. Hosting service based on Git
  2. More than just source control for you code
  3. Free and paid options

Getting your machine ready?

What you need

  1. Download Git from https://git-scm.com/downloads
  2. An editor Visual Studio Code
  3. GitHub account

Setting up your machine

  1. download and install git on your machine
  2. git config --global user.name "mreddy"
  3. git config --global user.email "t.mallikarjunreddy@gmail.com
  4. git config --edit --global
